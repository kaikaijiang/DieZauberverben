## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/die-zauberverben.git
   cd die-zauberverben
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally

Start the development server:
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

### Building for Production

Build the app for deployment:
```bash
npm run build
```
The output will be in the `dist` folder.

## Deployment

This project is configured to deploy automatically to **GitHub Pages** using GitHub Actions.

1. Push your code to the `main` branch.
2. The "Deploy to GitHub Pages" workflow will run.
3. Your site will be live at `https://YOUR_USERNAME.github.io/die-zauberverben/`.

## Privacy

- No personal data is collected.
- No cookies are used (except strictly necessary local storage for session settings like sound).
- Designed for child safety and classroom use.
